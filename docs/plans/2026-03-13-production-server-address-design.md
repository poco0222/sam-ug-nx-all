# 正式服务器地址切换设计

## 背景

当前仓库中的前端环境变量与 Qt 宿主默认编制人接口地址仍指向测试服务器 `192.168.19.110`。本次目标是将仓库默认地址统一切换为正式服务器 `192.168.19.100`，避免开发、打包与宿主默认配置不一致。

## 设计决策

### 改动范围

- `web-app/.env.development`
- `web-app/.env.production`
- `qt-host/src/bridge.cpp`
- `README.md`

### 方案选择

采用“统一修改默认正式地址”的方案：

- 前端开发环境默认地址改为 `http://192.168.19.100:8080`
- 前端生产环境默认地址改为 `http://192.168.19.100:8080`
- Qt 宿主编制人接口默认地址改为 `http://192.168.19.100:8080/UGApi/UGApiController/getBianChengUserInfo`

这样可以保证开发态、生产构建态和宿主兜底配置都默认落到正式服务器。

## 验证策略

- 静态检查修改后的目标文件内容
- 全文搜索 `192.168.19.110` 与 `192.168.19.100`，确认默认入口已完成切换
